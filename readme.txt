Get complicated and dizzy UTM url very short and simple.

== Description ==

Light WordPress plugin to translate very short URL to sophsticated UTM URL for Google Analytics 4.
Must work with the plugin: Google Analytics for WooCommerce

Add any type of UTM code you would like to measure ROI:
* Campaign source
* Campaign medium
* Campaign name
* Campaign term
* Campaign content

You can contribute by simple [Forking on GitHub](https://github.com/singleview-co-kr/singleview-google-analytics-utm-shortener).

== Installation ==

Install the plugin by:

1. Upload `singleview-google-analytics-utm-shortener` directory to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Go to `UTM shortener` page and add shorten URL.

== Frequently Asked Questions ==

= What is UTM code? =

A UTM (Urchin Tracking Module) code is a snippet of code that's added to the end of a URL to track and measure the performance of online marketing campaigns.

= Why is UTM code important? =

UTM codes are important because they help you: Understand audience behavior, Identify spend that drives high ROI, Optimize performance, and Generate business growth.
